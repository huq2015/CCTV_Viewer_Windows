# 电视浏览器 - Windows 版

基于 Electron 技术栈的 Windows 桌面电视直播浏览器，支持 CCTV 央视频道及全国地方卫视的在线观看。

---

## 项目概述

- 本项目为 Android 版「电视浏览器」基于Electron 框架开发的 Windows 适配移植版本，内置 Chromium 内核，通过 WebView 组件加载央视网、央视频官方网页直播页面，仅实现公开网页端电视频道在线浏览播放等主要功能。
- 项目初衷为闲置电脑生活化改造、替代传统电视盒子使用，主打客厅简易观影场景；因原版网页端适配遥控器体验不佳、操作步骤繁琐，本项目专为适配老人遥控器简易操作而开发。
- 特别声明：本项目仅做公开官方网页页面聚合浏览用途，无破解、截取、盗播、篡改官方内容等行为，所有视频资源及版权均归对应官方平台所有；项目仅作个人学习、非盈利自用交流，使用者可自行开源源码本地打包，自行承担下载、使用及二次分发的全部责任，请勿用于商业及违规用途。

### 核心特性

- **41 个电视频道**：20 个央视频道 + 21 个地方卫视
- **键盘遥控操作**：支持方向键换台、快捷键控制
- **自动全屏播放**：加载完成后自动进入全屏模式
- **页面精简加速**：自动屏蔽无关元素，提升加载速度
- **频道记忆功能**：自动保存上次观看频道
- **可自定义设置**：字体大小、换台方式等

### 进程职责

**主进程 (Main Process)**
- 创建和管理 BrowserWindow
- 注册全局键盘快捷键
- 管理应用生命周期
- 通过 electron-store 读写配置
- 配置 Session 请求头（User-Agent、Referer 等）
- 处理响应头（移除 X-Frame-Options、CSP 限制）

**渲染进程 (Renderer Process)**
- 渲染 UI 覆盖层（频道列表、菜单、设置）
- 嵌入 `<webview>` 标签加载电视网页
- 执行 JavaScript 注入（自动全屏、页面精简）
- 处理用户交互

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动应用（开发模式）

```bash
npm start
```

应用将以全屏模式启动，默认加载 CCTV-1 综合频道。

### 3. 打包构建

```bash
# 打包为可执行文件
npm run package

# 生成 Windows 安装程序
npm run make

# 仅生成 Windows 平台安装包
npm run make:win
```

打包输出目录：`out/make/squirrel.windows/x64/`

## 操作指南

### 键盘控制

| 按键 | 功能 | 说明 |
|------|------|------|
| `↑` | 上一个频道 / 打开频道列表 | 根据设置决定行为 |
| `↓` | 下一个频道 / 打开频道列表 | 根据设置决定行为 |
| `Enter` | 打开频道列表 | 显示所有频道选择界面 |
| `M` | 打开/关闭底部菜单 | 显示功能菜单栏 |

| `R` | 刷新页面 | 重新加载当前频道 |
| `F5` | 刷新页面 | 与 R 功能相同 |

### 频道列表操作

1. 按 `Enter` 打开频道列表
2. 使用 `↑` / `↓` 在频道间移动
3. 使用 `←` / `→` 切换央视频道/地方频道分类
4. 按 `Enter` 确认选择并换台
5. 按 `Esc` 关闭列表

### 底部菜单功能

按 `M` 键打开底部菜单，包含以下功能：

| 功能 | 说明 |
|------|------|
| 刷新页面 | 重新加载当前频道网页 |
| 播放/暂停 | 控制视频播放状态 |
| 切换全屏 | 切换视频全屏/窗口模式 |
| 放大页面 | 网页内容放大 10% |
| 缩小页面 | 网页内容缩小 10% |
| 打开设置 | 打开设置面板 |

### 设置选项

打开设置面板后，可调整以下选项：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用直接换台 | 按上下键直接换台，不弹出列表 | 关闭 |
| 换台抽屉直接返回 | 频道列表中按 Esc 直接关闭 | 开启 |
| 换台抽屉字体 | 频道列表字体大小（较小/适中/偏大/特大） | 适中 |

## 实现方案详解

### WebView 视频播放

应用核心通过 `<webview>` 标签加载电视直播网页：

1. **页面加载**：设置 `webview.src` 为频道 URL
2. **DOM 就绪**：注入 `FastLoading` 脚本，清空无关图片、脚本和 DIV 元素
3. **加载完成**：注入 `AutoFullscreen` 脚本，自动点击全屏按钮并设置音量

### JavaScript 注入

**FastLoading 脚本**
- 清空所有 `<img>` 标签的 `src` 属性
- 清空包含关键词（login、index、daohang 等）的脚本
- 清空特定 class（newmap、newtopbz 等）的 DIV 内容
- 递归执行，直到检测到全屏按钮出现

**AutoFullscreen 脚本**
- 检测全屏按钮（`#player_pagefullscreen_yes_player` 或 `.videoFull`）
- 自动触发点击事件
- 设置视频音量为 100%
- 每 16ms 轮询检测，直到成功

### 键盘事件处理

主进程通过 `webContents.on('before-input-event')` 监听键盘事件，通过 IPC 将事件转发到渲染进程，实现全局快捷键响应。

### 配置持久化

使用 `electron-store` 实现配置持久化：
- `currentChannelId`：上次观看的频道 ID
- `settings.directChannelChange`：直接换台开关
- `settings.directBack`：直接返回开关
- `settings.textSize`：字体大小设置

## 安全设计

- **进程隔离**：`nodeIntegration: false`，渲染进程无法直接访问 Node.js API
- **上下文隔离**：`contextIsolation: true`，预加载脚本与页面脚本隔离
- **IPC 白名单**：仅允许预定义的安全通道进行通信
- **WebView 分区**：使用独立 `partition`，隔离存储和 Cookie

## 常见问题

### Q: 为什么某些频道无法播放？
A: 请确保网络连接正常，且能够访问 `tv.cctv.com` 和 `yangshipin.cn`。应用已配置标准浏览器请求头（User-Agent、Referer 等）以模拟真实浏览器访问。如果仍无法播放，请检查：
1. 是否能正常访问央视网/央视频网站
2. 防火墙是否阻止了相关域名
3. 控制台是否有网络请求错误

### Q: 视频没有自动全屏？
A: 可能是网页结构更新导致选择器失效。请检查控制台日志，或尝试手动按菜单中的「切换全屏」按钮。

### Q: 如何退出全屏？
A: 按 `Esc` 键或 `B` 键，然后选择菜单中的「切换全屏」功能。

### Q: 安装包体积过大？
A: Electron 应用包含完整的 Chromium 内核，体积通常在 100-200MB 之间，这是正常现象。
